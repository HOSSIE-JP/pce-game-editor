PC Engine Core セットアップ 確認待ち

PCE Game Editor は llvm-mos-sdk に絞って HuCard / CD-ROM2 を扱います。画像インポートはエディター内蔵の PCE 変換を使用します。EmulatorJS は Test Play 用です。PCE-CD の IPL / System Card はライセンス上同梱せず、ユーザーが所有するファイルを指定します。

CD-ROM2 設定

CD-ROM2 Test Play / ビルド補助に使う IPL と System Card ROM を設定します。どちらも同梱できないため、所有するファイルを指定してください。

IPL は System Card BIOS ではありません。所有する PCE-CD / CD-ROM2 の ISO/CUE/BIN から先頭データセクタを抽出し、data/tools/pce-cd/ipl/ に保存します。
